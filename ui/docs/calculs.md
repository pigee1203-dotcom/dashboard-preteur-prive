# Calculs (référence)

## Intérêts (comme Margill)
Intérêt période = Solde début × Taux annuel × (jours / 365)

## Paiement
Paiement total = Paiement prêt + Frais adhésion
Paiement prêt = Capital remboursé + Intérêt

## Solde contrat
Solde fin = Solde début − Capital remboursé

## Souffrance (modèle prêteur privé)
Souffrance = Paiement manqué + Frais NSF + Frais report (si applicable)
Souffrance n’affecte pas le solde du contrat.
