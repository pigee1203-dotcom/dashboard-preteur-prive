# Actions (procédures simples)

## 1) Reporter un paiement
1. Cliquer l’icône “Reporter”
2. Choisir la nouvelle date
3. Le système calcule le frais (ex: 25$) et affiche un aperçu
4. Confirmer

## 2) NSF
1. Cliquer “Ajouter NSF”
2. Confirmer le paiement manqué + frais NSF
3. La souffrance augmente
4. Le contrat reste inchangé

## 3) Répartir souffrance
1. Choisir “Répartir”
2. Choisir X paiements
3. Aperçu des nouveaux montants
4. Confirmer
